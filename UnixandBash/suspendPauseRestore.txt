Suspend Pause or Restore some process.

01 - Ctrl + C
Finish the first plan process

02 - Ctrl + Z
Suspend the first plan process

03 - bg 
return all processes that are suspended

04 - jobs
list all the processes suspended and in second plan

05 - fg %1
return the first process suspended

06 - kill %1
Finish the first suspended process

07 - ps
List all active processes

08 - kill x
Finish the process of number x
